This guide uses the **Nine-Layer Pyramid Architecture** as its main thread, systematically covering the complete AI agent ecosystem:

```

Layer 4  Context (Global Information Carrier)
Layer 3  Prompt (Human-Machine Instruction Constraints)
Layer 2  LLM (Intelligent Reasoning Core)
Layer 1  Token (Physical Computing Foundation)
```
